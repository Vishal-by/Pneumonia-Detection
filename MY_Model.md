## Project Description

This project aims to detect pneumonia from chest X-ray images using a Convolutional Neural Network (CNN), a deep learning technique widely used for image classification tasks. Pneumonia is a lung infection that can cause serious health complications if not diagnosed early. The goal of this project is to develop an automated system that can assist in identifying pneumonia from chest X-ray images.

The model is trained using a dataset containing two classes: Normal and Pneumonia. During preprocessing, all images are resized and normalized to ensure consistent input for the neural network. The CNN automatically extracts important features from the X-ray images through convolution and pooling operations. These features are then passed through fully connected layers to classify the image as either Normal or Pneumonia.

The workflow of the model begins with image input, followed by preprocessing, feature extraction using convolutional layers, dimensionality reduction using pooling layers, and final classification using dense layers. The output layer produces a prediction indicating whether pneumonia is detected or not.

The model was trained for multiple epochs and achieved high training accuracy, demonstrating its ability to learn relevant patterns from chest X-ray images. The generated accuracy and loss graphs were used to evaluate model performance. This project demonstrates how Artificial Intelligence and Machine Learning can be applied in healthcare to support disease diagnosis and medical image analysis.
