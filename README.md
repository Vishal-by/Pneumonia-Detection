# Pneumonia Detection from Chest X-Ray Images using CNN

## Overview

This project uses a Convolutional Neural Network (CNN) to classify chest X-ray images as Normal or Pneumonia.

## Technologies Used

* Python
* TensorFlow
* NumPy
* Matplotlib
* Google Colab

## Dataset

Pediatric Chest X-ray Pneumonia Dataset

## Methodology

1. Data preprocessing and normalization
2. CNN model development
3. Model training and evaluation
4. Prediction of chest X-ray images

## Results

* Training Accuracy: Approximately 97%
* Validation Accuracy: Approximately 74%

## Output

The model predicts whether a chest X-ray image belongs to:

* Normal
* Pneumonia

## Future Scope

* Improve accuracy using Transfer Learning
* Deploy as a web application
* Support detection of multiple lung diseases
